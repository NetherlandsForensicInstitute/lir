class lir.transform.pipeline.LoggingPipeline(steps: list[tuple[str, lir.transform.Transformer | Any]], output_file: os.PathLike, include_batch_number: bool = True, include_labels: bool = True, include_fields: list[str] | None = None, include_steps: list[str] | None = None, include_input: bool = True)

Bases: lir.transform.pipeline.Pipeline

A pipeline that writes debugging output to a CSV file.

Initialization

Initialize a new LoggingPipeline instance.

Parameters:

  • steps – the steps of the pipeline as a list of (name, module) tuples.

  • output_file – the name of the generated output file

  • include_batch_number – (bool) whether the zero-indexed batch number should be included in the output, e.g. for cross-validation (defaults to True)

  • include_labels – (bool) whether the labels should be included, if available (defaults to True)

  • include_fields – a list of the names of extra fields to include (defaults to none)

  • include_steps – a list of steps to include (defaults to all)

  • include_input – (bool) whether to write the input features (defaults to True)
