class lir.aggregation.Aggregation

Bases: abc.ABC

Base representation of an aggregated data collection.

Other classes may extend from this class.

abstractmethod report(data: lir.aggregation.AggregationData) → None

Report that new results are available.

Parameters:

data – a named tuple containing the results

close() → None

Finalize the aggregation; no more results will come in.

The close method is called at the end of gathering the aggregation(s) to ensure files are closed, buffers are cleared, or other things that need to finish / tear down.

class lir.aggregation.SubsetAggregation(aggregation_methods: list[lir.aggregation.Aggregation], category_field: str)

Bases: lir.aggregation.Aggregation

Aggregation method that manages data categorization.

A separate aggregation method is used for each category.

Initialization

Initialize the subset aggregation method.

Parameters:

  • aggregation_methods – a list of methods to aggregate results by category

  • category_field – the name of the category field

report(data: lir.aggregation.AggregationData) → None
