Quickstart
----------

Get the code and install dependencies:

```shell
git clone -b practitioner_guide git@github.com:NetherlandsForensicInstitute/lir.git
pdm sync
```

Commands available:
- `pdm notebook`: launch the notebook in a browser
- `pdm clean`: reset the notebook
- `pdm test`: run unit tests
